# Mac

## 基础环境

### 常用命令
- 打开目录
```
open /
```
- 默认Apache
```
安装目录：/Library/WebServer
管理：sudo apachectl start|stop
```
- 推荐Apache
```
配置：/usr/local/etc/httpd
管理：sudo brew services start httpd
```
